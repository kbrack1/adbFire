Llama is an app that tells a device to perform tasks when certain conditions are met. It's very handy in controlling the AFTV.

It' very irritating to have to drill down seven or eight steps to start sideloaded apps on the AFTV. If you install ikono TV or Classic TV from the Amazon Store, Llama can hijack it, so when you press the the program icon, your chosen Media Center (Kodi,XBMC,SPMC) will start

This way you have the Media Center on the "Recent" menu of the AFTV. You may see the program app flash briefly before the Media Center software starts. You can also choose to replace the hijacked program's icon with the Media Center icon.

Click the "Install Llama" checkbox to install Llama.

Program Choice:

  • iKono TV
  • Classic TV
Media Center Choice
  • Kodi
  • XBMC
  • SPMC
  • TVMC
Llama Events
  • Boot the AFTV to the Media Center This starts the chosen Media Center at boot.
  • Link the Media Center to chosen Program: ikono TV or Classic TV
  • Both. Start the Media Center when the AFTV boots and link to the selected program, Ikono TV or Classic TV
  • No action (the default) Do not adjust Llama settings
Icon Choice
  • Replace the Program icon with Media Center icon
  • Restore the default Program icon
  • No action (the default)

Note:After installing Llama and selecting an event, you must run Llama and:

  • Press the FTV remote menu button
  • Choose Import/Export Data
  • Import from USB Storage
Llama will import the settings generated with adbFire.

If you have questions or comments, please email jocala@jocala.com or visit one of the forums linked below. adbFire produces a log file, adbfire.log, with each run. Include it in your email or post if you're having problems.
