Llama is an app that tells a device to perform tasks when certain conditions are met. It's very handy in controlling the AFTV. The "Llama Options" button opens a dialog that offers different Llama related events:

The "Install Llama" Click this checkbox to install Llama.

1. Boot the AFTV to Kodi
Starts Kodi when the AFTV boots.

2. Link Kodi to ikono TV
It's very irritating to have to drill down seven or eight steps to start sideloaded apps on the AFTV. If you install the app "ikono TV" from the Amazon Store, Llama can hijack it, so when you press the ikono icon, Kodi starts. This way you have Kodi on the "Recent" and "Apps" menu of the AFTV. You may see the ikono TV app flash briefly before Kodi starts.

3. Both
Kodi starts when the AFTV boots AND is linked to the ikono TV app.

4. No action
Do not adjust Llama settings.

The next four options relate to the ikono TV icon.

1. Replace ikono TV icon with Kodi icon
2. Replace ikono TV icon with Kodi icon
3. Restore default ikono TV icon
4. No action (the default)

Note: 1. After installing Llama and selecting an event, you must run Llama and select "import event from USB". The events that adbFire installs start Kodi/XBMC. They will need to be edited to start variants like SPMC.

If you have questions or comments, please email jocala@jocala.com or visit one of the forums linked below. adbFire produces a log file, adbfire.log, with each run. Include it in your email or post if you're having problems.
