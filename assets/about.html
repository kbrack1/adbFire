adbFire Overview

adbFire is a companion program for Amazon Fire TV & Android Kodi and its forks.

adbFire allows you to install (sideload) apps to your Android device without the bother of installing the Android SDK.

Additionally, it allows you copy files to and from your device, backup and restore Kodi's data, and more.

The upper left Current Device combobox contains the names of your Android devices. This dropdown combobox is empty when you first run adbFire. Once you add devices, you choose a device from this menu in order to interact with it. This combobox is empty when you first start adbFire. It fills as you add device records.

The buttons associated with the device dropdown menu are:

Edit: Edit the selected device.

New: Display a blank device data entry screen (New USB devices are also added via a double-click in the Connected Devices listbox, see below).

Delete: Delete the selected device.

Connect: Connect the selected device..

Disconnect: Disconnect the selected device.

The Connect and Disconnect buttons are disabled if the selected device is connected via USB.

Connections are attempted automatically for most adbFire functions. For example, should the Backup button be pressed and the selected device uses IP for a connection, adbFire will make the connection without a button press.

The Connected Devices listbox shows all current adb connections.

The Status listbox shows the device status (device/emulator/offline,etc).

Double-clicking a USB device ID in the Connected Devices listbox opens a new device record with the ID filled in. This allows you to easily create a device record for USB connections

The Restart ADB button: Does refresh of connected devices

The Stop ADB button stops the ADB server.

The Preset directories dropdown menu allows you to select a directory to push files to, pull files from or delete files. All Kodi directories are included as well as sdcard and extUsb

The Ad Hoc Directory entry field allows entry of any directory you have access to on the device for push/pull/delete.

The Push button allows selection of a file on the PC to be pushed to the selected directory on the connected device.

The Pull button allows selection of a file on the device to be pulled from the selected directory on the connected device.

The Del button allows selection of a file on the device for deletion.

The Clear button clears the Ad Hoc directory selection.

The lower buttons are:

Install apk: Installs an apk file.

Uninstall apk: Uninstalls an apk file.

Screen Shot: Takes a screen shot of whatever is on your device's TV screen.

ADB Shell: Opens a console on the connected device. Much like telnet.

Backup: Create a Kodi backup.

Restore: Restore a Kodi backup (made by adbFire).

Console: Opens a console for command-line entry of adb commands.

View Log: View a log of your current or previous adbFire session. Copy the log to the system clipboard for pasting into messages,email,etc.

Llama options: Use Llama to make Kodi appear on Launcher "Recent" menu

Push Remote: Push a remote control XML file to Kodi

The following buttons Require Root

Mount: Mount USB drives/sticks

Unmount:Unmount USB drives/sticks

SSH:SSH into selected device

SFTP:SFTP into selected device

FS R/O:Make the file system read-only

FS R/W:Make the file system read-write
