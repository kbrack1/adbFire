Install SSH
This option installs the Dropbear SSH server on your device. This allows you to open a secure shell on your Fire TV, somewhat like adb shell. Among many other things, it allows you to copy files to your device as root using the scp command and act as a backup entryway into your system in the case of a problem with adb.

The sftp option is also present, which allows an ftp-like connection to the Fire TV

SSH can be made persistant by checking the "SSH at boot" option in adbFire's preferences dialog. If you choose not to start SSH at boot it can be started via SFTP/SSH buttons or via the command line:

Pressing either SSH or SFTP will open a connection to Fire TV.

From the Fire TV ADB Shell you have three commands to control the SSH server:

  • sshstart: start the ssh server
  • sshstop: stop the ssh server
  • sshstatus: give the server status

For Windows users the puTTY ssh/sftp clients are pre-installed, OS X and Linux users have default ssh/sftp clients.
