Mount Network Shares

The Mount Network Shares function allows you to mount four shares of types NFS or CIFS

Each share consists of:
Selection checkbox

IP Address of share

Mount point for share

Protocol checkbox
(checked=NFS unchecked=CIFS)

There are five buttons:
Mount - Mount any share with the selection checkbox checked

Unmount - Unmount any share with the selection checkbox checked

Save - Save all shares

Script - Write an init.d script to mount checked shares at boot

Cancel - Exit the dialog with no action taken.

Shares and mount points are entered like so:

192.168.1.2 (your share IP)
/path/to/your/share
/path/to/mountpoint

This is a ROOT function. This will not work on unrooted devices.
